This project focused on designing Organicus, a mobile application that allows users to browse and purchase organic food products in a simple and visually appealing way. The goal was to create a clean, modern, and user-friendly interface that supports a smooth and intuitive user journey from discovery to product selection.

Key Features

* Category-based navigation using pill components for easy browsing
* Product listings with bold, standout product names and clear calorie information
* Visually structured product detail pages to help users make informed decisions
* Consistent spacing, typography, and color usage across all screens
* Minimal and clean UI style to reflect a healthy and organic brand identity

Screens Included

* Splash Screen (brand introduction for Organicus)
* Home Screen (entry point with featured categories and products)
* Category Navigation (pill-style filters for quick access)
* Product Listing (organized display of available items)
* Product Details (in-depth view with product name, calories, and description)

Design Approach

The design of Organicus was centered around usability, clarity, and visual hierarchy. A strong emphasis was placed on making product names stand out, ensuring users can quickly scan and identify items.

Consistency was maintained across all screens by using a structured spacing system and a cohesive color palette that reflects freshness and health. The flow was designed to guide users naturally from browsing categories to exploring product details without confusion.

Tools Used

* Figma

Outcome

Organicus is a cohesive mobile app UI that demonstrates a strong understanding of user flow, component usage, and structured design thinking. The project highlights the ability to design not just individual screens, but a connected user experience that feels intuitive, modern, and purposeful.
